To set up a new Flow node you will need to complete the following steps:

1. [Provision](/node-operation/node-setup) the machine on which your node will run.

2. Generate a node identity by following the [bootstrap](/node-operation/node-bootstrap) process.

3. [Start](/node-operation/node-bootstrap#step-2---getting-ready-to-start-your-flow-node) your node!
